# 🌦️ Weather App

A simple and responsive Weather App built using React.

## 🚀 Features
- 🔍 Search weather by city name
- 🌡️ Shows temperature, humidity, and conditions
- ⚡ Fast and clean UI

## 🛠️ Tech Stack
- React
- Vite
- OpenWeather API

## 📦 Installation

1. Clone the repository
2. Install dependencies

npm install

3. Run the app

npm run dev
